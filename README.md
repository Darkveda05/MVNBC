MVNBC (Multivendor Network Backup Config) is a Bash-based CLI tool for automating configuration backups across multi-vendor network devices like router/switch & firewall.

Setup Requirements:

    sudo apt install sshpass -y
    chmod +x MVNBC.sh
    ./MVNBC.sh

The repository is written entirely in Shell/Bash and appears to be a straightforward automation utility for network device management.
