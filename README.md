# MVNBC (Multivendor Network Backup Config)

MVNBC is a Bash-based CLI tool for automating configuration backups across multi-vendor for network devices.

Current working products:
1) cisco
2) cisco ASA
3) H3C
4) Arista
5) Aruba CX
6) Mikrotik


Requirements:
1) Linux (Ubuntu/debian/casaos)
2) git
3) zsh

Installation Step:

    sudo apt install git
    sudo apt install zsh
    sudo apt install sshpass -y
    git clone https://github.com/Darkveda05/MVNBC.git
    cd MVNBC
    chmod +x MVNBC.sh
    ./MVNBC.sh

The repository is written entirely in Shell/Bash and appears to be a straightforward automation utility for network device management.
